---
title: About ReviewPrompt
description: Product positioning and repository purpose.
---

ReviewPrompt is an open registry of AI-assisted peer-review prompts, workflows, and agent skills.

The project is designed for reviewers, authors, lab groups, meta-reviewers, and open-source contributors who want reusable review methods instead of scattered prompt snippets.

The first version focuses on content quality, structured metadata, and installable workflow packaging. Later versions can add community review queues, richer search, prompt builders, and evaluation benchmarks.
