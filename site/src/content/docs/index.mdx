## 为什么做这个项目

ReviewPrompt 将分散的论文审稿提示词转化为可复用的审稿基础设施。第一版聚焦结构化资源：带元数据的提示词、带步骤的工作流，以及带安全说明的可安装 Skills。

## 信任模型

- 开源，并通过 Git 管理版本。
- 提示词元数据包含适用场景、状态、来源、许可证和限制。
- Skill 页面区分纯提示词资源和可安装 Agent 工作流。
- 安全说明明确：AI 输出是决策辅助，不替代人类判断。
