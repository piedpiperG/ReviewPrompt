title: 贡献提示词
description: 向 ReviewPrompt 提交人工智能论文审稿提示词。
---

## 可以提交什么

ReviewPrompt 当前只接受提示词投稿。一个有用的提交应该说明：

- 支持完整论文还是 Rebuttal。
- 面向作者还是审稿人。
- 预期输入和输出。
- 已知限制和安全风险。
- License、来源和贡献者署名。

## 提交路径

- 使用 GitHub Issue Form 提交提示词。

## 审核标准

- 任务和目标用户清晰。
- 输出格式关注证据。
- 不编造引用或会议政策。
- 包含可用示例或模板。
- License 兼容。
