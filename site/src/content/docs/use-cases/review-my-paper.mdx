## Recommended Path

1. Open the Full Paper Review Workflow.
2. Copy the Full Structured Peer Review prompt or install the matching skill.
3. Provide the manuscript text, abstract, method, experiments, and venue rubric if available.
4. Run a second score calibration pass before finalizing the review.
