## Why This Exists

ReviewPrompt turns scattered peer-review prompts into reusable review infrastructure. The first version focuses on structured resources: prompts with metadata, workflows with steps, and installable skills with safety guidance.

## Trust Model

- Open source and versioned in Git.
- Prompt metadata includes use cases, status, source, license, and limits.
- Skill pages separate prompt-only resources from installable agent workflows.
- Safety guidance treats AI output as decision support, not a replacement for human judgment.
