## Recommended Path

1. Use the Pre-submission Self Review Workflow.
2. Run Reviewer Attack Simulation to expose likely objections.
3. Use Methodology Critique and Reproducibility Checklist for evidence gaps.
4. Turn the output into a revision checklist and prioritize by submission risk.
