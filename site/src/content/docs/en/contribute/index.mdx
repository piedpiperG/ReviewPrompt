## What to Submit

ReviewPrompt currently accepts prompt submissions only. A useful submission should explain:

- Whether it supports full paper review or rebuttal.
- Whether it is designed for authors or reviewers.
- The expected input and output.
- Known limitations and safety concerns.
- License, source, and contributor attribution.

## Submission Paths

- Use the GitHub Issue Form to submit a prompt.

## Review Criteria

- Clear task and target user.
- Evidence-aware output format.
- No fabricated citations or venue policies.
- Usable examples or templates.
- Compatible license.
