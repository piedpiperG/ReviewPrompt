---
title: Roadmap
description: Product phases for ReviewPrompt.
---

## Phase 0

Publish the personal open-source library: GitHub repo, GitHub Pages website, curated prompts, workflows, skills, and contribution entry.

## Phase 1

Add community submission flow, review queue, contributor pages, prompt status, version history, and license policy.

## Phase 2

Make the project skill-first: richer Skill Registry, install guides, ZIP downloads, changelogs, and safety levels.

## Phase 3

Build an interactive prompt builder for non-technical users.

## Phase 4

Add evaluation assets: example paper set, output comparison, human rating, failure cases, and model comparison.
