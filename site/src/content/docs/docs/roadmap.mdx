---
title: 路线图
description: ReviewPrompt 的产品阶段。
---

## Phase 0

发布个人开源库：GitHub repo、GitHub Pages 网站、精选 Prompts、Workflows、Skills 和贡献入口。

## Phase 1

加入社区投稿流程、审核队列、贡献者页面、Prompt 状态、版本历史和 License 政策。

## Phase 2

让项目转向 skill-first：更完整的 Skill Registry、安装指南、ZIP 下载、changelog 和安全等级。

## Phase 3

为非技术用户构建交互式 Prompt Builder。

## Phase 4

加入评测资产：示例论文集、输出对比、人类评分、失败案例和模型对比。
