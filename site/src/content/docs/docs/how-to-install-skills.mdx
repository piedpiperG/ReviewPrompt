---
title: 如何安装 Skills
description: 将 ReviewPrompt Skills 安装到本地 Agent 工作流。
---

## Codex

项目级 Skills 可以放在 `.agents/skills`：

```bash
mkdir -p .agents/skills
cp -r skills/full-paper-review .agents/skills/
```

## Claude Code

个人 Skills 可以放在 `~/.claude/skills`：

```bash
mkdir -p ~/.claude/skills
cp -r skills/full-paper-review ~/.claude/skills/
```

## 通用 Agent 用法

如果你的工具不直接支持 Skills，可以打开对应 `SKILL.md`，手动复用其中的 workflow、guardrails 和输出格式。
