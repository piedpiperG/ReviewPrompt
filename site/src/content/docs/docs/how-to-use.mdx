## 按任务选择

先从任务出发，而不是从 Prompt 名字出发。你可以使用首页推荐器，也可以选择以下路径：

- 审一篇投稿论文：完整论文审稿流程。
- 准备自己的论文投稿：投稿前自审稿流程。
- 回应审稿意见：Rebuttal 模拟流程。
- 汇总多份审稿意见：Meta-review 综合流程。

## 提供证据

当你提供与问题相关的论文部分时，AI 辅助审稿效果最好。尽量包含摘要、方法、实验、相关工作、局限和会议 rubric。

## 将输出视为草稿

输出是决策辅助。人类审稿人和作者仍然需要对判断、公平性、保密性和最终措辞负责。
